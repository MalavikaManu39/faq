# 📖 JavaScript FAQ Accordion

A simple and interactive FAQ Accordion project built using **HTML, CSS, and JavaScript**.  
This project demonstrates basic **DOM manipulation** and **event handling** in JavaScript.

---

## ✨ Features

- Show/Hide answers on click
- Interactive FAQ layout
- Clean and responsive design
- Beginner-friendly project

---

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript

---

## 📚 JavaScript Concepts Used

- `querySelectorAll()`
- `addEventListener()`
- `nextElementSibling`
- `style.display`

---

## ⚙️ How It Works

- JavaScript selects all FAQ questions
- A click event is added to each question
- Clicking a question toggles the visibility of its answer

---

## 🚀 Future Improvements

- Smooth animations
- Open only one FAQ at a time
- Add `+` / `−` icons
- Dark mode support

---

## 📷 Output

An interactive FAQ section where users can click questions to expand or collapse answers.
