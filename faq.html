JavaScript FAQ

What is JavaScript?
JavaScript is a programming language used to make webpages interactive.
What is DOM?
DOM stands for Document Object Model. It allows JavaScript to access and modify HTML elements.
What is querySelector()?
querySelector() is used to select HTML elements using CSS selectors.
What is addEventListener()?
addEventListener() is used to attach events like click, mouseover, and keypress to elements.
What is classList.toggle()?
classList.toggle() adds a class if it is missing and removes it if it already exists.
